# Litematic_V7_To_V6_DynamicLibrary
在尽可能保留方块、方块实体、实体等其它信息的情况下，  
降低投影数据版本(投影降级)，从 V7(MC1.20.5+) 转换到 V6(MC1.20.4-)   

本项目为[Litematic_V7_To_V6](https://github.com/chenjunfu2/Litematic_V7_To_V6)的库版本，用于提供[litematica-extra](https://github.com/shuangshun/litematica-extra)调用  
使用的NBT库为：[NBT_CPP](https://github.com/chenjunfu2/NBT_CPP/)
